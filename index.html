CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can apply styles directly to elements, class names or IDs.
  • We can define the width and height of any element
  • Just because you can make things really colourful with CSS, doesn't mean you should!

Git

  • Git can be confusing, but practice and repetition helps
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Adds all modified files
  • git commit -m: commit staged changes
  • git pull origin main: Make sure current branch matches the main
  • git push origin feature/name-of-feature: push all file changes
